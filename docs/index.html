渐进式CSS库, ONLY CSS!

Jvase专为设计师或者后端开发者提供的一套CSS样式库，旨在快速、高效地构建项目.

下载Jvase 文档

Jvase 是一套灵活、渐进式的CSS库，开发者可以很方便地通过添加类来实现不同风格的界面。

为什么使用Jvase?

⇜

响应式

能最大程度适应不同终端、不同浏览器的显示.

⌤

适应性

通过添加类，可DIY各种风格，能适应不同类型的组件.

☍

模块化

每个模块独立成一个文件，可以单独使用.
